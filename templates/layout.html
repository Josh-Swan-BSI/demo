Using the Standards Explorer and Layer Explorer Tools

Lookup Table

The lookup table can be used to discover the metadata of Designated standards and all types of references. Checking the row boxes on the left-hand side will load the standard into the Layer Explorer tool on the right. Please note only designated standards can currently be loaded into the Layer Explorer. The global search will search all pages and cells and will highlight what it finds. If search is pressed, it will filter the rows to just display the relevant content. This search is case-insensitive and also can search for more than one term using a comma separating the terms. Pressing clear will remove all search terms, checked boxes and blue-highlighted cells.

Each column has its own filter box to allow more specific filtering. The user has the option to switch between case-sensitive and case-insensitive, and by default they are set to be case-sensitive. As well as searching text, other commands can be used here:

Date Searching Examples

  • Equals: =2005-12-12
  • Newer than: >2005-12-12
  • Equal or newer than: >=2005-12-12
  • Older than: <2005-12-12
  • Equal or older than: <=2005-12-12

Layer Explorer

The Layer Explorer allows to investigate the relionship between designated standards, their references, and the references of the references. The tool is powered by the datatable and can load multiple designated standard 'trees' into the display window. To discover metadata of the plot points, hover your mouse over a point. The chart comes equipped with some controls in the top right corner which allows the user to do the following:

  • Download a png of the plot in its current view
  • Zoom - draw a box area to zoom into (can also be done with mouse wheel)
  • Panning - which allows the image to be moved around
  • Box select - for highlight points in a box area
  • Lasso select - for highlight points in a freeform area
  • Zoom in/out
  • Autoscale - sizes the graph to fit all plots in an ideal view
  • Reset axis - resets the graph to the default view
